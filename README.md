# My Website

This project is a minimal starter template for building a website using the Content module with Nuxt.js. It's designed to provide a simple foundation for creating content-rich websites with ease.

## Tech Modules in Use

This project utilizes several Nuxt.js modules to enhance its functionality. Below is a list of these modules along with links to their documentation for further exploration:

- **@nuxt/content**: A powerful module to add a Git-based Content Management System (CMS) to Nuxt. It allows you to write your content in Markdown, JSON, YAML, or CSV files and query it through a MongoDB like API. [Nuxt Content Documentation](https://content.nuxtjs.org/)
- **@nuxt/image**: An image optimization module with support for responsive images, lazy loading, and image placeholders. [Nuxt Image Documentation](https://image.nuxtjs.org/)
- **@nuxt/fonts**: Easily add and manage custom fonts in your Nuxt project. [Nuxt Fonts Documentation](https://github.com/nuxt-modules/fonts)
- **@nuxt/ui**: Provides a set of UI components for Nuxt projects. [Nuxt UI Documentation](https://ui.nuxtjs.org/)
- **@nuxt/eslint**: Integrates ESLint into Nuxt projects for code quality and consistency. [Nuxt ESLint Documentation](https://eslint.nuxtjs.org/)
- **@antfu/eslint-config**: An ESLint configuration package that provides an opinionated code style guide, heavily used and recommended for Vue and Nuxt projects for maintaining code quality. [GitHub Repository](https://github.com/antfu/eslint-config)

These modules collectively contribute to the robustness, performance, and ease of development of the project.

## Features

- Minimal setup: Get started quickly without the hassle of a complicated setup process.
- Content module: Leverage the power of [Nuxt Content](https://content.nuxtjs.org/) to manage and display your website's content.
- Markdown support: Write your content in Markdown for easy formatting and management.
- Image optimization: Utilize [@nuxt/image](https://image.nuxtjs.org/) for automatic image optimization, improving load times and performance.
- Custom fonts: Easily incorporate custom fonts into your project with [@nuxt/fonts](https://github.com/nuxt-modules/fonts), enhancing your site's typography.
- Enhanced UI components: Access a suite of UI components designed for Nuxt projects via [@nuxt/ui](https://ui.nuxtjs.org/), helping to speed up development.
- Code quality assurance: Ensure high code quality and consistency across your project with integrated ESLint support through [@nuxt/eslint](https://eslint.nuxtjs.org/) and [@antfu/eslint-config](https://github.com/antfu/eslint-config).
